# 🌍 Voyago Travels

> **Every Journey Becomes A Story**

A modern, responsive, and visually engaging travel agency website developed as part of the **MANTRA 2026 Summer School Frontend Website Assignment**. Voyago Travels showcases premium travel destinations, luxury tour packages, an interactive gallery, and a seamless booking experience while following modern frontend development and SEO best practices.
